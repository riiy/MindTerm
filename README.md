# Mind Terminal

## Overview

## Installation

Ensure you are using Python 3.11 or later (but less than 4.0).

``` shell
pip install mindterm --user
```

## Usage

``` shell
mt
```

## Testing

Tests for the app are included using [pytest](https://docs.pytest.org/). To run the tests, simply execute:

```bash
pytest
```
